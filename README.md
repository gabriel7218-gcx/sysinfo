# sysinfo

Minimal and portable system information script written in pure shell.

## Features

- Displays hostname, kernel, CPU, architecture, uptime, RAM usage and storage
- Compatible with OpenWrt (overlayfs)
- Compatible with standard Linux distributions
- Works on WSL2
- No external dependencies
- BusyBox compatible

## Compatibility

Tested on:

- OpenWrt (MIPS / overlayfs)
- x86_64 Linux
- WSL2
- Termux (Mobile)
